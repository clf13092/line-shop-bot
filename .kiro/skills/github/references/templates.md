# Issue/PR テンプレート

## Issue テンプレート

### バグ報告

```markdown
## 概要
<!-- バグの簡潔な説明 -->

## 再現手順
1. 
2. 
3. 

## 期待する動作
<!-- 本来どうなるべきか -->

## 実際の動作
<!-- 実際に何が起きたか -->

## 環境
- OS: 
- ブラウザ/バージョン: 
- アプリバージョン: 

## スクリーンショット
<!-- あれば添付 -->

## 補足
<!-- その他の情報 -->
```

### 機能要望

```markdown
## 概要
<!-- 機能の簡潔な説明 -->

## 背景・課題
<!-- なぜこの機能が必要か -->

## 提案する解決策
<!-- どのように解決したいか -->

## 代替案
<!-- 他に検討した方法があれば -->

## 補足
<!-- その他の情報 -->
```

### タスク

```markdown
## 概要
<!-- タスクの説明 -->

## 完了条件
- [ ] 
- [ ] 
- [ ] 

## 関連Issue
<!-- #123 など -->
```

### 質問・相談

```markdown
## 質問内容
<!-- 質問の詳細 -->

## 背景
<!-- なぜこの質問に至ったか -->

## 調べたこと
<!-- 自分で調査した内容 -->
```

## Pull Request テンプレート

### 機能追加

```markdown
## 概要
<!-- 変更内容の簡潔な説明 -->

## 関連Issue
Closes #

## 変更内容
- 
- 
- 

## 動作確認
- [ ] ローカルで動作確認済み
- [ ] テスト追加/更新済み
- [ ] ドキュメント更新済み

## スクリーンショット
<!-- UIの変更があれば添付 -->

## レビュー観点
<!-- 特に見てほしいポイント -->
```

### バグ修正

```markdown
## 概要
<!-- 修正内容の簡潔な説明 -->

## 関連Issue
Fixes #

## 原因
<!-- バグの原因 -->

## 修正内容
<!-- どのように修正したか -->

## 動作確認
- [ ] バグが再現しないことを確認
- [ ] 既存機能への影響がないことを確認
- [ ] テスト追加済み
```

### リファクタリング

```markdown
## 概要
<!-- リファクタリング内容 -->

## 目的
<!-- なぜリファクタリングするか -->

## 変更内容
- 
- 

## 動作確認
- [ ] 既存テストがパスすることを確認
- [ ] 機能に変更がないことを確認
```

### ドキュメント更新

```markdown
## 概要
<!-- 更新内容 -->

## 変更ファイル
- 
- 

## 確認事項
- [ ] リンク切れがないことを確認
- [ ] 誤字脱字チェック済み
```

### 依存関係更新

```markdown
## 概要
<!-- 更新するパッケージ -->

## 変更内容
| パッケージ | 変更前 | 変更後 |
|-----------|--------|--------|
|           |        |        |

## 理由
<!-- なぜ更新するか（セキュリティ、機能追加など） -->

## 動作確認
- [ ] ビルドが通ることを確認
- [ ] テストがパスすることを確認
- [ ] 破壊的変更への対応済み
```

---

## ラベル例

| ラベル | 用途 |
|--------|------|
| `bug` | バグ報告 |
| `feature` | 機能要望 |
| `enhancement` | 改善 |
| `documentation` | ドキュメント |
| `question` | 質問 |
| `help wanted` | 助けが必要 |
| `good first issue` | 初心者向け |
| `priority: high` | 優先度高 |
| `priority: low` | 優先度低 |
| `wontfix` | 対応しない |
| `duplicate` | 重複 |
